## Como executar

- Clone o repositório
- Instale as dependências com `yarn`
- Inicie o servidor de dados com `yarn server`
- Em um novo terminal, inicie o servidor com `yarn dev`
- Acesse a aplicação em `localhost:8080` no navegador de sua preferência
